RESOLUSI GAMA
1. Konsisten untuk belajar coding dan programming minimal 2 jam sehari
2. Membuat todolist untuk membuat jadwal dan roadmap untuk mencapai tujuan menjadi web developer
3. Mulai melakukan habit positif setiap hari seperti membaca buku, olahraga, journaling untuk menaikkan value diri.
4. Membeli dan menyelesaikan online course untuk menambah skill coding.
5. Ikuti tantangan / challenge di coding platform seperti LeetCode atau Hackerrank untuk mengasah logika pemrograman.
